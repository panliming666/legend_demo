## 🎮 特色玩法

### 灵宠系统
- **8阶灵宠**：骨灵→灵狼→灵熊→灵虎→仙鹤→云龙→凤凰→天将
- **进化升级**：战斗获得经验，满级进化
- **召唤战斗**：最多同时召唤多个灵宠助战

### 炼丹系统
- **5类丹药**：治疗、法力、增益、突破、特殊
- **20种配方**：从回春丹到化神丹
- **炼丹等级**：炼制提升炼丹师等级

### 秘境副本
| 副本 | 层数 | Boss | 奖励 |
|-----|------|------|------|
| 新手试炼 | 3层 | 无 | 经验+灵草 |
| 灵气洞穴 | 5层 | 无 | 灵芝+灵石 |
| 白骨坑 | 7层 | 骨魔 | 妖丹+灵玉 |
| 恶魔巢穴 | 10层 | 恶魔领主 | 金丹草 |
| 仙人陵墓 | 15层 | 仙灵守护者 | 元婴果 |
| 龙巢 | 20层 | 远古青龙 | 龙血+神玉 |
