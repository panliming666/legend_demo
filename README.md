# Legend Demo - 八方旅人美术风格 + 传奇玩法

## 项目简介

这是一个将**八方旅人（Octopath Traveler）的HD-2D美术风格**与**传奇类游戏的核心玩法**相结合的独立游戏项目。

## 核心特性

### 🎨 美术风格
- HD-2D渲染效果（像素角色 + 3D场景）
- 八方旅人风格的光影效果
- 动态天气系统

### ⚔️ 游戏玩法（传奇风格）
- **职业系统**：战士/法师/道士（可扩展）
- **装备爆装**：传奇经典的随机掉落机制
- **等级成长**：击杀敌人获取经验升级
- **行会系统**（计划中）
- **攻城战**（计划中）

## 技术栈

- **游戏引擎**：Godot 4.2
- **开发语言**：GDScript
- **目标平台**：PC（Windows/Linux）+ 移动端

## 核心系统详解

### 1. 战士职业（已实现）
- **基础属性**：HP/MP/攻击/防御
- **移动控制**：8方向移动
- **攻击系统**：普通攻击 + 技能框架
- **成长系统**：升级提升属性

### 2. 装备系统（传奇风格）
- **稀有度分级**：
  - 普通（白色）- 100%权重
  - 优秀（绿色）- 50%权重
  - 稀有（蓝色）- 20%权重
  - 史诗（紫色）- 5%权重
  - 传说（橙色）- 1%权重

- **装备类型**：
  - 武器（主攻）
  - 护甲（主防+HP）
  - 头盔（防御+MP）
  - 靴子（速度+防御）
  - 饰品（全属性）

### 3. 掉落系统
- **经验掉落**：基于敌人等级
- **金币掉落**：随机波动 + 等级加成
- **装备掉落**：传奇经典概率公式
  - 基础掉落率：30%
  - 传说装备：0.1%
  - 史诗装备：0.5%
  - 稀有装备：2%
  - 优秀装备：10%

### 4. 敌人AI系统
- **检测范围**：200像素
- **追击范围**：自动追击玩家
- **攻击范围**：30像素
- **随机巡逻**：脱战后巡逻

## 如何运行

1. **安装Godot 4.2**
   ```bash
   # 从官网下载：https://godotengine.org/download
   ```

2. **打开项目**
   - 启动Godot编辑器
   - 选择"打开项目"
   - 导航到`legend_demo`文件夹
   - 点击"打开"

3. **运行游戏**
   - 按F5或点击"运行"按钮

## 控制方式

- **移动**：方向键（↑↓←→）
- **攻击**：空格键
- **技能**：Q/E/R（预留）
