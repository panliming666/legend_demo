# 三清问道 - 少年修仙传奇

## 项目简介

**三清问道**是一款融合**少年修仙题材**与**传奇核心玩法**的独立游戏。

> "混沌初分，三清立道。少年有缘，拜入仙门，问道长生。"

游戏背景设定在修仙世界，玩家扮演一位初入修仙界的少年，拜入三清天尊座下，修习仙法，斩妖除魔，最终证道成仙。

## 三宗职业

### 🌊 玉清宗 · 法修（原法师）
**天尊**：元始天尊  
**道号**：玄元弟子  
**主修**：法力（魔法攻击）  

**特点**：
- 操控天地法则，释放元素法术
- 远程输出，范围AOE
- 高爆发伤害

**独门法术**：
- 灵火术（火系单体）
- 雷法·引雷诀（召唤天雷）
- 玄冰咒（冰系减速）
- 九天神雷（范围雷法）

**修炼境界**：
玄元童子 → 玄元真人 → 玄元尊者 → 玄元仙师 → 玄元真君

---

### 🌿 上清宗 · 符修（原道士）
**天尊**：灵宝天尊  
**道号**：灵宝弟子  
**主修**：道法（道术攻击）  

**特点**：
- 符箓咒术，召唤灵兽
- 辅助治疗，团队辅助
- 策略性强

**独门法术**：
- 回春符（治疗）
- 腐骨符（持续毒伤）
- 召唤·骨灵（召唤骷髅）
- 召唤·天将（高级召唤）

**修炼境界**：
灵宝童子 → 灵宝真人 → 灵宝尊者 → 灵宝仙师 → 灵宝真君

---

### ⚔️ 太清宗 · 剑修（原战士）
**天尊**：道德天尊  
**道号**：道德弟子  
**主修**：武力（物理攻击）  

**特点**：
- 炼体修剑，近战无敌
- 高防御，肉身成圣
- 操作简单

**独门法术**：
- 基础剑法
- 剑气斩（远程剑气）
- 疾风剑（快速连斩）
- 剑阵·千剑归宗（范围剑雨）

**修炼境界**：
道德童子 → 道德真人 → 道德尊者 → 道德仙师 → 道德真君

## 游戏系统

### 特色系统
- **炼丹系统**：炼制突破丹、治疗丹
- **炼器系统**：打造法宝、洗炼词条
- **召唤系统**：召唤灵宠助战
- **御剑飞行**：金丹期解锁飞行

### 宗门系统
- **拜师三清**：选择宗门，确定职业
- **宗门大战**：争夺资源矿
- **秘境副本**：获得功法、丹药、法宝

### 道友切磋
- **清修模式**：不攻击他人
- **切磋模式**：友好比试
- **问道模式**：全力厮杀
- **恶念系统**：红名惩罚

## 核心设计理念

### 1. 本土化创新
- 将传统战法道融入修仙背景
- 三清道统对应三大职业
- 境界系统替代等级系统

### 2. 差异化设定
- 法修：远程法术，高爆发
- 符修：召唤辅助，策略性
- 剑修：近战肉搏，高防御

### 3. 深度养成
- 境界突破系统
- 法宝祭炼系统
- 炼丹炼器系统
- 灵宠召唤系统
