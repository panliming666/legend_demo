# 三清问道 - 单机修仙游戏

> 混沌初分，三清立道。少年有缘，拜入仙门问道长生。

**单机修仙RPG**，融合传奇玩法与Roguelike元素。

---

## 核心玩法

### 🎮 游戏模式

| 模式 | 描述 |
|-----|------|
| **剧情模式** | 5章主线剧情，修仙证道 |
| **无尽爬塔** | 无限层数，挑战极限 |
| **Roguelike** | 随机地图+遗物，每局不同 |
| **挑战模式** | 限时/生存/无伤等挑战 |
| **多周目** | 最高10周目，难度递增 |

### ⚔️ 战斗系统

- **三宗职业**：法修/符修/剑修
- **技能树**：攻击/防御/辅助/终极
- **灵宠系统**：8阶进化
- **坐骑系统**：御剑飞行
- **自动战斗**：挂机刷图

### 🎒 装备系统

- **法宝**：12槽位
- **炼器**：打造强化
- **炼丹**：20种丹药
- **时装**：外观收集
- **翅膀**：飞升系统

### 📊 进度系统

- **境界**：炼气→化神
- **成就**：50+成就
- **称号**：20+称号
- **收集**：全图鉴
