# 三清问道

> 混沌初分，三清立道。少年有缘，拜入仙门问道长生。

---

## 宗门

| 宗门 | 天尊 | 职业 | 特点 |
|-----|------|------|------|
| 玉清宗 | 元始天尊 | 法修 | 远程法术，范围AOE |
| 上清宗 | 灵宝天尊 | 符修 | 召唤灵宠，团队辅助 |
| 太清宗 | 道德天尊 | 剑修 | 近战肉搏，高防御 |

## 核心玩法

- **灵宠进化**：8阶灵宠
- **坐骑系统**：8种坐骑
- **翅膀系统**：7阶翅膀
- **世界Boss**：四圣兽定时刷新
