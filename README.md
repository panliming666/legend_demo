# 三清问道 - 少年修仙传奇

## 项目简介

**三清问道**是一款融合**少年修仙题材**与**传奇核心玩法**的独立游戏。

> "混沌初分，三清立道。少年有缘，拜入仙门问道长生。"

---

## 🌟 三大宗门

### 🌊 玉清宗 · 法修
**天尊**：元始天尊  
**特点**：远程法术，范围AOE，高爆发  

**法术**：灵火术 → 雷法·引雷诀 → 九天神雷 → 天地法相

---

### 🌿 上清宗 · 符修
**天尊**：灵宝天尊  
**特点**：符咒召唤，团队辅助，策略性强  

**法术**：回春符 → 召唤·骨灵 → 召唤·天将 → 灵宝金光咒

---

### ⚔️ 太清宗 · 剑修
**天尊**：道德天尊  
**特点**：近战肉搏，高防御，剑气外放  

**法术**：基础剑法 → 剑气斩 → 御剑术 → 剑阵·千剑归宗

---

## 🛠️ 技术栈

- **引擎**：Godot 4.2
- **语言**：GDScript
- **平台**：PC + 移动端

## 📂 项目结构

```

├── docs/
│   ├── SANQING_WORLD_SETTINGS.md  # 世界观设定

│   ├── character_class.gd    # 职业系统

```

## 🎯 控制方式

- **移动**：鼠标左键按住
- **跑动**：鼠标右键按住
- **技能**：数字键 1-4
