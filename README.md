## 🎮 核心玩法

### 灵宠进化
骨灵→灵狼→灵熊→灵虎→仙鹤→云龙→凤凰→天将

### 坐骑进化
骏马→灵鹿→灵狼→猛虎→仙鹤→雄狮→飞龙→凤凰

### 世界Boss时间
- 青龙 12:00 东方云海
- 朱雀 18:00 南方火山
- 白虎 20:00 西方荒原
- 玄武 22:00 北方冰原
