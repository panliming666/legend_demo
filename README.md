# Legend Demo - 八方旅人美术风格 + 传奇玩法

## 项目简介

这是一个将**八方旅人（Octopath Traveler）的HD-2D美术风格**与**传奇类游戏的核心玩法**相结合的独立游戏项目。

## 核心特性

### 🎨 美术风格（HD-2D）
- ✅ 程序化像素纹理生成（草地/石头/木头/水面）
- ✅ 角色精灵生成器（战士/史莱姆等）
- ✅ 后期处理着色器（扫描线/像素完美）
- ✅ 3D场景支持（光照/雾效）
- ⏳ HD-2D渲染管线配置

### ⚔️ 游戏玩法（传奇风格）
- ✅ **三职业系统**：战士/法师/道士
- ✅ **装备爆装**：传奇经典的随机掉落机制
- ✅ **等级成长**：击杀敌人获取经验升级
- ✅ **存档系统**：自动保存/手动保存
- ✅ **关卡系统**：5个关卡+Boss战
- ⏳ 行会系统
- ⏳ 攻城战

## 技术栈

- **游戏引擎**：Godot 4.2
- **开发语言**：GDScript
- **目标平台**：PC（Windows/Linux）+ 移动端

## 职业系统详解

### 1. 战士 ⚔️
- **定位**：近战物理输出
- **属性**：高HP/高防御
- **技能**：普通攻击
- **特点**：操作简单，适合新手

### 2. 法师 🔮
- **定位**：远程魔法输出
- **属性**：高MP/高魔攻
- **技能**：
  - 火球术：单体高伤害
  - 冰盾：增加防御
  - 雷电术：AOE伤害
- **特点**：爆发伤害，操作要求高

### 3. 道士 🐍
- **定位**：召唤/辅助
- **属性**：均衡发展
- **技能**：
  - 召唤骷髅：召唤2个骷髅战斗
  - 治愈术：恢复HP
  - 毒云：持续AOE伤害
  - 祝福：增益攻击和防御
- **特点**：策略性强，召唤流玩法

## 装备系统（传奇风格）

### 稀有度分级
- **普通**（白色）：100%权重
- **优秀**（绿色）：50%权重
- **稀有**（蓝色）：20%权重
- **史诗**（紫色）：5%权重
- **传说**（橙色）：1%权重

### 装备类型
- 武器（主攻）
- 护甲（主防+HP）
- 头盔（防御+MP）
- 靴子（速度+防御）
- 饰品（全属性）

## 如何运行

1. **安装Godot 4.2**
   - 从官网下载：https://godotengine.org/download

2. **克隆仓库**
   ```bash
   git clone https://github.com/panliming666/legend_demo.git
   ```

3. **打开项目**
   - 启动Godot编辑器
   - 选择"打开项目"
   - 导航到`legend_demo`文件夹
   - 点击"打开"

4. **运行游戏**
   - 按F5或点击"运行"按钮

## 控制方式

- **移动**：方向键（↑↓←→）
- **攻击**：空格键
- **技能**：预留（Q/E/R）
