# 三清问道 - 游戏世界观设定

## 一、世界背景

### 天地初开
混沌初分，清气上升为天，浊气下沉为地。三清天尊顺应天道，各立道统，广收门徒，教化众生。

**三清者：**
- **玉清元始天尊**：掌管天地法则，传授玄门法术
- **上清灵宝天尊**：掌管符箓丹药，传授灵宝秘术
- **太清道德天尊**：掌管修身炼体，传授武道真经

### 修炼体系
少年拜入三清门下，修习各派道法，最终证道成仙。

## 六、宗门系统

### 拜师条件
- 完成"三清问道"任务
- 选择其中一派拜师
- 不可更改（可转修，代价大）

### 宗门职位
| 职位 | 权限 | 条件 |
|-----|------|------|
| 宗主 | 全部权限 | 宗门创建者 |
| 长老 | 管理权限 | 贡献值前10 |
| 内门弟子 | 宗门福利 | 活跃度达标 |
| 外门弟子 | 基础福利 | 普通成员 |

### 宗门大战
- **报名时间**：每周
- **战斗时间**：周末
- **胜利奖励**：资源矿、专属称号

## 七、道友系统（PK）

### 切磋模式
| 模式 | 说明 | 影响 |
|-----|------|------|
| 清修 | 不攻击他人 | 安全 |
| 切磋 | 友好比试 | 无惩罚 |
| 论道 | 认真对战 | 掉落物品 |
| 问道 | 全力厮杀 | 加恶念值 |

### 恶念系统
- 击杀白名道友：恶念+100
- 恶念>0：黄名
- 恶念>500：红名
- 红名死亡：掉落法宝

## 十、开局选择

### 三清问道
少年初入修仙界，三清天尊各赐机缘：

**元始天尊赐：**《玉清玄法录》→ 成为**法修**
**灵宝天尊赐：**《上清灵宝经》→ 成为**符修**  
**道德天尊赐：**《太清剑道篇》→ 成为**剑修**
