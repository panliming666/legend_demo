# 传奇世界游戏设定

## 一、职业系统

### 1. 战士
- **主属性**：物理攻击
- **特点**：高HP、高物理防御、近战输出
- **技能**：
  - 基础剑法：普通攻击
  - 刺杀剑术：单体高伤害
  - 半月弯刀：前方扇形AOE
  - 野蛮冲撞：冲锋并击退敌人
  - 烈火剑法：大幅提升攻击力
  - 逐日剑法：远程剑气攻击

### 2. 法师
- **主属性**：魔法攻击
- **特点**：高MP、高魔法伤害、远程AOE
- **技能**：
  - 火球术：单体火系伤害
  - 雷电术：单体雷系伤害
  - 抗拒火环：推开周围敌人
  - 地狱火：直线AOE
  - 冰咆哮：范围冰系伤害+减速
  - 魔法盾：吸收伤害
  - 灭天火：火系终极技能
  - 流星火雨：大范围AOE

### 3. 道士
- **主属性**：道术攻击
- **特点**：辅助、召唤、持续伤害
- **技能**：
  - 治愈术：恢复HP
  - 精神力战法：增加道术攻击
  - 施毒术：持续毒伤害
  - 召唤骷髅：召唤骷髅战士
  - 召唤神兽：召唤神兽
  - 群体治愈术：范围恢复
  - 困魔咒：束缚敌人
  - 召唤月灵：召唤月灵攻击

## 二、属性系统

### 基础属性
| 属性 | 说明 | 影响 |
|-----|------|-----|
| HP | 生命值 | 生存能力 |
| MP | 魔法值 | 技能消耗 |
| 物理攻击 | 近战伤害 | 战士主要输出 |
| 魔法攻击 | 魔法伤害 | 法师主要输出 |
| 道术攻击 | 道术伤害 | 道士主要输出 |
| 物理防御 | 减少物理伤害 | 防御能力 |
| 魔法防御 | 减少魔法伤害 | 魔法抗性 |
| 准确 | 命中率 | 攻击命中率 |
| 敏捷 | 闪避率 | 被攻击闪避 |
| 攻击速度 | 攻击频率 | DPS |
| 移动速度 | 移动速率 | 机动性 |

### 隐藏属性
- **暴击率**：触发暴击的概率
- **暴击伤害**：暴击时的伤害倍数
- **生命恢复**：每秒恢复HP
- **魔法恢复**：每秒恢复MP
- **幸运**：影响掉落和暴击
- **反射**：反弹部分伤害

## 三、装备系统

### 装备槽位（12个）
| 槽位 | 名称 | 主要属性 |
|-----|------|---------|
| 头盔 | Helmet | 防御、生命 |
| 项链 | Necklace | 魔法、准确 |
| 左手镯 | Bracelet_L | 攻击、攻速 |
| 右手镯 | Bracelet_R | 攻击、攻速 |
| 左戒指 | Ring_L | 攻击、暴击 |
| 右戒指 | Ring_R | 攻击、暴击 |
| 腰带 | Belt | 生命、敏捷 |
| 靴子 | Boots | 防御、移速 |
| 武器 | Weapon | 主攻击 |
| 衣服 | Armor | 主防御 |
| 宝石1 | Gem_1 | 额外加成 |
| 宝石2 | Gem_2 | 额外加成 |

### 装备稀有度
| 稀有度 | 颜色 | 属性倍率 | 掉落率 |
|-------|------|---------|--------|
| 普通 | 白色 | 1.0x | 70% |
| 高级 | 绿色 | 1.2x | 20% |
| 稀有 | 蓝色 | 1.5x | 7% |
| 卓越 | 金色 | 2.0x | 2% |
| 史诗 | 紫色 | 2.5x | 0.8% |
| 传说 | 橙色 | 3.0x | 0.2% |

### 装备等级
- 每件装备有等级要求
- 等级越高的装备属性越强
- 可通过强化提升装备等级

### 装备强化
- 强化等级：+1 ~ +10
- 强化成功：属性提升10%
- 强化失败：等级不变（传统版本掉级，简化版不降）
- 成功率：每级降低10%

### 套装效果
- 2件套：基础属性+5%
- 4件套：基础属性+10%，特殊效果
- 6件套：基础属性+20%，终极效果

## 四、怪物系统

### 怪物等级
| 等级 | 类型 | HP | 攻击 | 掉落 |
|-----|------|----|----|------|
| 1-10 | 小怪 | 低 | 低 | 普通 |
| 11-20 | 精英 | 中 | 中 | 高级 |
| 21-30 | 首领 | 高 | 高 | 稀有 |
| 31-40 | 精英首领 | 极高 | 极高 | 卓越 |
| 41-50 | Boss | 超高 | 超高 | 史诗/传说 |

### 怪物类型
- **物理型**：高物理攻击，低魔防
- **魔法型**：高魔法攻击，低物防
- **均衡型**：各项属性平均
- **召唤型**：会召唤小怪
- **精英**：强化属性
- **Boss**：特殊技能

### 刷新机制
- 普通怪：60秒刷新
- 精英怪：5分钟刷新
- Boss：30分钟刷新
- 世界Boss：固定时间刷新

### 危险区

- 副本地图：队伍专属
- Boss房间：限时挑战

## 六、PK系统

### PK模式
| 模式 | 说明 | 影响 |
|-----|------|------|
| 和平 | 不可攻击玩家 | 安全 |
| 全体 | 可攻击所有玩家 | 增加PK值 |
| 行会 | 可攻击非行会成员 | 行会战 |
| 编组 | 可攻击非队友 | 组队 |
| 善恶 | 仅攻击红名玩家 | 惩罚恶人 |

### PK值系统
- 击杀白名玩家：PK值+100
- PK值>0：黄名
- PK值>500：红名
- 红名死亡：掉落装备

### 红名惩罚
- 不能进入安全区
- 被NPC攻击
- 死亡掉落概率增加
- PK值每小时-10

## 七、行会系统

### 行会创建
- 需要等级30级
- 需要100万金币
- 创建者成为会长

### 行会职位
| 职位 | 权限 |
|-----|------|
| 会长 | 全部权限 |
| 副会长 | 邀请、踢人、公告 |
| 长老 | 邀请成员 |
| 成员 | 无管理权限 |

### 行会等级
- 1级：50人上限
- 2级：100人上限，需要10万行会资金
- 3级：150人上限，需要30万行会资金
- 4级：200人上限，需要60万行会资金
- 5级：300人上限，需要100万行会资金

### 行会技能
| 技能 | 效果 | 升级条件 |
|-----|------|---------|
| 凝聚力 | 经验+5% | 行会资金 |
| 财富之眼 | 金币+5% | 行会资金 |
| 守护之力 | 防御+5% | 行会资金 |
| 战争号角 | 攻击+5% | 行会资金 |

## 八、攻城战

### 报名条件
- 行会等级≥3级
- 报名费：50万金币
- 报名行会数：最多10个

### 攻城时间
- 报名：周一至周五
- 准备：周六
- 战斗：周日20:00-22:00

### 据点系统
- 北门、南门、东门、西门
- 中央皇宫
- 占领据点获得积分

### 胜利条件
- 积分最高的行会获胜
- 或占领皇宫30分钟

### 奖励
- 城主行会：每日税收、专属称号
- 参与者：经验、金币、道具

## 九、经济系统

### 金币获取
- 击杀怪物掉落
- 出售装备给NPC
- 完成任务奖励
- 参与活动

### 金币消耗
- 购买装备/道具
- 装备强化
- 技能学习
- 行会相关

### 交易系统
- 玩家间自由交易
- 摆摊系统
- 寄售系统

## 十、等级系统

### 等级上限
- 当前版本：60级
- 后续开放：100级

### 升级经验
| 等级 | 所需经验 |
|-----|---------|
| 1-10 | 100-1000 |
| 11-20 | 2000-10000 |
| 21-30 | 20000-100000 |
| 31-40 | 200000-1000000 |
| 41-50 | 2000000-10000000 |
| 51-60 | 20000000-50000000 |

### 升级奖励
- HP/MP提升
- 属性点（可选分配）
- 技能点
- 新技能解锁

## 十一、技能系统

### 技能学习
- 完成任务获得技能书
- 使用技能书学习技能
- 部分技能需前置技能

### 技能升级
- 使用技能获得熟练度
- 熟练度满后技能升级
- 升级后伤害提升

### 技能类型
- 主动技能：需要手动释放
- 被动技能：自动生效
- 辅助技能：增益效果
- 召唤技能：召唤单位

## 十二、日常活动

### 每日任务
- 击杀怪物x20
- 收集材料x5
- 完成副本x1

### 限时活动
- 世界Boss（12:00, 18:00, 22:00）
- 押镖（全天可接）
- 宝图（全天可挖）
- 经验副本（周末双倍）

### 周常活动
- 行会战
- 攻城战
- 排位赛
